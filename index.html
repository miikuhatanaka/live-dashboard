🌐 クラウド同期設定

GitHub GistにデータをURLするとダッシュボードをウェブ公開でき、他の人が閲覧できるようになります。
Gist ID のみ入力すると閲覧専用モード。 GitHubトークン（gistスコープ）も入力するとオーナーモード（編集・同期可能）になります。

📖 初回設定手順（クリックで展開）
【オーナー向け】
1. gist.github.com で新しいGistを作成。ファイル名を data.json、内容を [] にして「Create secret gist」。
2. GistのURLにある32文字のIDをコピー（例: https://gist.github.com/username/ここがID）。
3. github.com/settings/tokens で「gist」スコープのみのPATを作成。
4. 上のフォームに入力して「💾 設定を保存」→「🔄 今すぐ同期」でローカルデータをGistへ転送。
5. このHTMLを index.html に改名し、ファイル内の const DEFAULT_GIST_ID = '7d5425efae6b412284a5e27841592f34' にGist IDを貼り付けて保存。
6. GitHubリポジトリに push → Settings > Pages > mainブランチで公開 → URLをシェア。

【閲覧者向け】
オーナーからURLを受け取ったら、そのままアクセスするだけで最新データが閲覧できます。
（Gist IDが埋め込まれていない場合は、Gist IDのみ入力して「💾 設定を保存」）
